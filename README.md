Descrição do Projeto

Este projeto é o primeiro passo para a construção de um jogo completo de **Batalha Naval**.  
Neste desafio, você irá:

- Criar um **tabuleiro 10x10** usando uma matriz (array bidimensional);
- Declarar dois navios de tamanho **3** usando vetores (arrays unidimensionais);
- Posicionar um navio **horizontalmente** e outro **verticalmente**;
- Garantir que os navios não ultrapassem os limites do tabuleiro e **não se sobreponham**;
- Exibir o tabuleiro com valores:
  - `0` → água  
  - `3` → parte do navio

Tudo isso com lógica simples e organizada, ideal para o nível novato.

---

Objetivos

- Trabalhar com **matrizes (2D)** e **vetores (1D)**.  
- Validar posições dentro dos limites do tabuleiro.  
- Posicionar navios de forma segura.  
- Aplicar loops aninhados (`for`) para exibir o tabuleiro.  
- Criar código limpo, comentado e organizado.

---

Estrutura do Programa

Tabuleiro (Matriz 10x10)
- Inicializado com **0**  
- Representa água  

Navios (Vetores 1D)
- Cada navio tem tamanho **3**  
- Valores do vetor = `3`  
- Um navio horizontal, outro vertical  

Regras de posicionamento
- Navios **não podem ultrapassar** o tabuleiro.  
- Navios **não podem se sobrepor**.  
- Coordenadas definidas diretamente no **código**.
